# Java Day 5 Tasks

This repository contains my **Java Day 5** practice programs developed using **Eclipse IDE**. The programs are created to improve my understanding of Java programming concepts through hands-on coding.

## Topics Covered

* Object-Oriented Programming (OOP)
* Classes and Objects
* Constructors
* Inheritance
* Polymorphism
* Method Overloading and Overriding
* Exception Handling
* User Input using Scanner
* Collections Framework
* Basic Java Programs

## Technologies Used

* Java
* Eclipse IDE
* JDK 17 (or your installed version)

3. Ensure the JDK is installed and configured.
4. Run the required Java file as a Java Application.
